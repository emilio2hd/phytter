# Pytter
Watcher for new tweets, built in Python 2.7.6
Tested on Xubuntu 14.04

# Requirements
* tweepy (3.2.0)
* python-dateutil (2.4.0)
* python-appindicator (12.10.1)
* python-gtk2 (2.24.0)
* pyinotify (0.9.5)
